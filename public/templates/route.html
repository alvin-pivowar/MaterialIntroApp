ngRoute

The ngRoute module provides routing and deeplinking services and directives for angular apps. AngularJS Documentation: ngRoute

$routeProvider

Used for configuring routes. AngularJS Documentation: $routeProvider

ngView

ngView is a directive that complements the $route service by including the rendered template of the current route into the main layout (index.html) file. Every time the current route changes, the included view changes with it according to the configuration of the $route service. AngularJS Documentation: ngView

$location

The $location service parses the URL in the browser address bar (based on the window.location) and makes the URL available to your application. Changes to the URL in the address bar are reflected into $location service and changes to $location are reflected into the browser address bar. AngularJS Documentation: $location
